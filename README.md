# Speech Insight Demo

End-to-end demonstrator for speech analysis on noisy construction-site audio.

**Goal:** transcribe, extract key fields, and optionally generate follow-up questions as audio.

## Stack
- Whisper ASR (speech-to-text)
- LLM-based info extraction
- FastAPI demo app
- Optional TTS (gpt-4o-mini-tts or Bark)
